# MazBasura - Prototipo

Prototipo académico de aplicación móvil para consultar rutas y horarios de recolección de basura en Mazatlán.

## Descripción

MazBasura es una propuesta de aplicación móvil enfocada en mejorar la comunicación entre el servicio de recolección de basura y los ciudadanos. La aplicación permite consultar información sobre rutas, horarios, alertas y direcciones favoritas.

## Objetivo

Diseñar un prototipo de aplicación móvil que facilite la consulta de rutas y horarios de recolección de basura, ayudando a los usuarios a organizarse mejor y estar informados.

## Herramientas utilizadas

- Figma
- Diseño de interfaces
- Análisis de requerimientos
- Documentación de software

## Funcionalidades propuestas

- Consulta de rutas de recolección
- Visualización de horarios
- Registro de direcciones favoritas
- Notificaciones o alertas
- Información del servicio
- Perfil de usuario

## Evidencias

Las capturas del prototipo se encuentran en la carpeta **Capturas**.
